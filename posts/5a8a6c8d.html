前言

本文是对于luohAPI的文档介绍，便于更方便的使用。

介绍

图片类

动漫图

获取随机的动漫类图片

接口地址  https://api.luoh.my.to/anime.php

请求方法 GET

返回格式 image json

请求参数说明

名称必填类型说明
type是string需要获取的图片类型 可选为:
first second third genshin mobile
其中后二者为
原神类 手机尺寸类
return否string指定返回类型，可选为:
image json

返回示例


返回参数说明

名称类型说明
statusstring状态码
imageurlstring图片链接 或 错误信息

请求示例 https://api.luoh.my.to/anime.php?type=first&return=image

 

头像

获取随机的头像类图片

接口地址  https://api.luoh.my.to/avatar.php

请求方法 GET

返回格式 image json

请求参数说明

名称必填类型说明
type是string需要获取的图片类型 可选为:
dm gf kt ka random
其分别对应
动漫 古风 卡通 可爱 随机
return否string指定返回类型，可选为:
image json

返回示例


返回参数说明

名称类型说明
status是string状态码
imageurl是string图片链接

请求示例 https://api.luoh.my.to/avatar.php?type=dm&return=image

 

猫

获取随机的猫类图片

接口地址  https://api.luoh.my.to/cat.php

请求方法 GET

返回格式 image json

请求参数说明

名称必填类型说明
return否string指定返回类型，可选为:
image json

返回示例


返回参数说明

名称类型说明
status是string状态码
imageurl是string图片链接

请求示例 https://api.luoh.my.to/cat.php?return=image

 

表情包

获取随机的表情包类图片

接口地址  https://api.luoh.my.to/emoticon.php

请求方法 GET

返回格式 image json

请求参数说明

名称必填类型说明
type是string需要获取的图片类型 可选为：
scymm ecybqb sej sxly sxlyhb cjm hj xmt mmckb gcmm lx lt
其分别为
三次元猫猫 二次元表情包 兽耳酱 塞西莉娅 塞西莉娅黑白 柴郡猫 滑稽 熊猫头 猫猫虫咖波 甘城猫猫 罗翔 龙图
return是string指定返回类型，可选为:
image json

返回示例


返回参数说明

名称类型说明
status是string状态码
imageurl是string图片链接

请求示例 https://api.luoh.my.to/emoticon.php?type=cjm&return=image
